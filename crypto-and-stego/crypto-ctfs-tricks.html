SMS Code

Multitap 通过重复数字 替换字母，这些数字由手机 键盘 上对应的键码定义（此模式在编写 SMS 时使用）。
例如：2=A，22=B，222=C，3=D...
您可以通过看到** 多个数字重复** 来识别此代码。

您可以在以下网址解码此代码：https://www.dcode.fr/multitap-abc-cipher

Bacon Code
